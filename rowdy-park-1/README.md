# Rowdy Park

Welcome to the Rowdy Park project! This innovative parking application is designed to enhance the parking experience for users, utilizing the vibrant colors of the University of Texas at San Antonio (UTSA).

## Project Structure

The project is organized as follows:

```
rowdy-park
├── src
│   ├── index.html          # Main HTML file for the website
│   ├── components          # Contains reusable HTML components
│   │   ├── header.html     # Header section with navigation and branding
│   │   └── footer.html     # Footer section with copyright and links
│   ├── styles              # CSS stylesheets
│   │   ├── vars.css        # CSS variables for UTSA colors
│   │   └── main.css        # Main styles for layout and typography
│   ├── scripts             # JavaScript files
│   │   └── app.js         # Application logic and interactivity
│   └── assets              # Assets like icons
│       └── icons           # Directory for icon files
├── public
│   └── manifest.json       # Web app manifest with metadata
├── package.json            # npm configuration file
├── .gitignore              # Files and directories to ignore by Git
└── README.md               # Project documentation
```

## Features

- **User-Friendly Interface**: Designed with a clean and intuitive layout.
- **Real-Time Parking Availability**: Users can check available parking spots in real-time.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **UTSA Branding**: Incorporates UTSA colors and branding elements throughout the application.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd rowdy-park
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```

## Usage

Once the server is running, open your browser and navigate to `http://localhost:3000` to view the Rowdy Park application. Explore the features and enjoy a seamless parking experience!
